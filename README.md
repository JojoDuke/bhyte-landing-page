# Bhyte Landing Page

A modern AI-native software company landing page built with Next.js, featuring an interactive navbar and multiple sections.

## Features

- **Modern Design**: Dark theme with interactive navigation
- **Responsive Layout**: Works on all devices
- **AI-Focused Sections**: Studio, Labs, and Intelligence pages
- **Interactive Navigation**: Hover animations and active states
- **Custom Font**: Ranade font from Fontshare
- **Color-Coded Sections**: Visual indicators for each section

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/studio` - AI model building and development tools
- `/labs` - Experimental AI research and features
- `/intelligence` - AI-driven insights and automation

## Built With

- [Next.js](https://nextjs.org) - React framework
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [Ranade Font](https://fontshare.com) - Custom typography
